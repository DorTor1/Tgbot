# Telegram-бот для 3x-ui

Бот регистрирует клиентов в панели [3x-ui](https://github.com/MHSanaei/3x-ui), хранит заявки в SQLite и выдаёт ссылки подписки.

## Быстрый старт

1. Python 3.11+ (рекомендуется).
2. Скопируйте переменные окружения и заполните своими значениями:

   ```bash
   cp .env.example .env
   ```

3. Установите зависимости и запустите:

   ```bash
   python3 -m venv .venv
   source .venv/bin/activate   # Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   python main.py
   ```

Подробности переменных — в комментариях в `.env.example`.

## Безопасность и публикация на GitHub

- **Не коммитьте** файл `.env`, базы `*.db` / `*.sqlite3`, каталог `.venv/` — они уже перечислены в `.gitignore`.
- В репозитории не должно быть реальных URL панели, токена бота, паролей и путей подписки: только шаблоны в `.env.example` и нейтральные значения по умолчанию в коде.
- Если `.env` или секреты **когда-либо попали в git** (в том числе в старых коммитах): снимите файл с индекса (`git rm --cached .env`), смените пароль панели и **отзовите токен бота** у [@BotFather](https://t.me/BotFather) и создайте новый — старый считать скомпрометированным.
- Перед `git push` имеет смысл проверить: `git grep -iE 'token|password|AAG[A-Za-z0-9_-]{30,}'` и убедиться, что в отслеживаемых файлах нет ваших секретов.
